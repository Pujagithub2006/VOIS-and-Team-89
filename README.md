# SilverCare  
## Proactive AI Platform for Elderly Assistance, Health Monitoring & Caregiver Support  
**VOIS Innovation Marathon 2.0 Project**

---

##🔹 About the Project

**SilverCare** is an AI-powered wearable healthcare monitoring system designed to ensure the safety, independence, and well-being of elderly individuals living alone.

It combines a **sensor-based wearable waistband**, **real-time monitoring**, **AI-driven detection**, and **instant caregiver alerts** to prevent serious incidents like falls, inactivity, or missed medication.

This repository contains the complete implementation including:
- Hardware integration
- AI logic
- Dashboards
- Alert system
- Medicine scheduling
- Project PPT documentation

---

## 🔹 Problem Statement

With the rising elderly population, many seniors live independently without continuous supervision, which leads to:

- Undetected falls and instability  
- Prolonged inactivity  
- Missed medication doses  
- Delayed emergency response  
- Increased risk of preventable injuries  

SilverCare provides a proactive monitoring solution to address these risks.

---

## 🔹 Objectives

- Monitor elderly movement and activity in real time  
- Detect falls and abnormal behavior  
- Send automated alerts to caregivers  
- Provide medicine reminders  
- Enable remote health supervision  

---

## 🔹 Solution Overview

SilverCare is a **wearable waistband device + AI platform** that:

- Collects sensor data continuously  
- Detects abnormal patterns using smart logic  
- Sends SMS alerts during emergencies  
- Provides dashboards for caregivers  
- Helps elderly users stay safe without restricting movement
